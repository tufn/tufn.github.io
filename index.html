FAQ

Questions, answered

What is Tufn?

Tufn is a minimalist all-in-one productivity app combining notes, tasks, a digital canvas, and calendar in a single clean interface built for deep focus.

When does it launch?

Tufn is currently 75% complete and targeting a Q1 2027 public release. Join the waitlist to be first in line and receive development updates.

What makes it different?

Unlike tools that try to do everything, Tufn is built around a clear philosophy: minimal interface, maximal focus. No feature bloat, no subscriptions, no ads.

Is Tufn free?

Yes, completely free. Productivity tools should be accessible to everyone — no freemium traps, no paid tiers, no hidden costs.

Which platforms?

Tufn will support Windows, macOS, Linux, iOS, and Android — plus a web version that runs in any modern browser.

Is my data secure?

All data is encrypted in transit and at rest. We collect zero personal data beyond what you explicitly provide. See our Privacy Policy for details.
